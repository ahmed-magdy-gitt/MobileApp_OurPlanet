## ğŸŒŒ Project Overview
**Our Planet** is an engaging educational Android application designed to explore the wonders of our Solar System. Built purely with **Kotlin**, the app offers users a visually appealing gallery of planets.

It demonstrates efficient list handling using `RecyclerView` with a `GridLayoutManager`, allowing users to navigate through planets and access rich, localized information (Arabic Content) about each celestial body's distance, temperature, and unique characteristics.
